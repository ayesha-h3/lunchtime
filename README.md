### Lunchtime!
A Visual Novel for the Girls Got Game Summer Jam themed to "the void." 

![title cover](image.png)

[try it here!](sleepymeep.itch.io/lunchtime)

Features:
- Original characters!
- Handwritten story with multiple endings
- Decisions that affect how your story goes
- Unique storytelling style

We used Ren'Py for our project as it was very easy to start in and springboard off of. We were able to collaborrate via GitHub to delegate parts of the scripting and worked as a team to create the necessary art (on a combination of Ibis Paint and Autodesk Sketchbook) as well.

All characters are original art by @brdy.e. Backgrounds by @sleepymeep (with the exception of one, the lunchtime ending, which is an asset from Magnific). Music from Pixabay.
